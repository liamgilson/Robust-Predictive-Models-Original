# Description

R source code for Gilson, Eskelson, Sattler, & O'Neill

# Files:

./00_Functions.R : Required functions

./01_AB_data_read.R : Reads Alberta Spruce Provenance Trials

./02_410_read.R : Reads 410 series (Ontario) Spruce Provenance Trials

./03_faib_read.R : Reads BC FAIB (PSP) database

./04_FIA_read.R : Reads FIA (USFS) databases for selected juristictions

./05_Sx_read.R : Reads BC Sx spruce provenance trial data

./10_data_merge.R : combines harmonized data with climate

./20_analysis.R : Variable selection, internal model validation

./30_mag_process : Reads MAGPlot database for external validation, performs validation using QC PSP data

./31_pg_trial_process : Reads and processes Prince George provenance trial data for validation

./32_faib_preds.R : performs projections using FAIB data to 2100

./33_pg_preds.R : performs projections using Prince George data to 2100

./40_graphics.R : manuscript graphics

./41_results_tables.R : manuscript tables
